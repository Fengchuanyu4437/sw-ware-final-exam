# sw-ware-final-exam
Board Game with 7 blue stone and 7 red stone

+---+---+---+---+---+
| B | B | B | B | B |
+---+---+---+---+---+
| B |   |   |   | B |
+---+---+---+---+---+
|   |   |   |   |   |
+---+---+---+---+---+
| R |   |   |   | R |
+---+---+---+---+---+
| R | R | R | R | R |
+---+---+---+---+---+

The winner is the player who first moves each of his or her stones to a square that was initially occupied by a stone of the opponent.
